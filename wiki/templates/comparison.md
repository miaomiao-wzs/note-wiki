# 对比页模板

这个模板用于 `wiki/<module>/comparisons/` 下的对比页面。

## 适用场景

- 两个或多个业务对象对比
- 不同模块差异对比
- 处理流程差异对比
- 方案优缺点对比

## 模板内容

```markdown
---
module: fico
kind: comparison
updated: 2026-07-06
source_count: 2
tags: []
---

# 对比主题

## 对比对象
- A：[[对象 A]]
- B：[[对象 B]]

## 结论先行
先写最重要的差异结论。

## 对比维度
| 维度 | A | B |
| --- | --- | --- |
| 适用场景 |  |  |
| 处理方式 |  |  |
| 优点 |  |  |
| 局限 |  |  |

## 关键差异
- 差异 1
- 差异 2

## 参考来源
- [[来源页 1]]
- [[来源页 2]]
```

## 使用原则

- 对比页要先给结论，再给细节。
- 对比维度要固定，方便以后横向扩展。
- 如果只是在讲一个对象本身，不要硬写成对比页。
