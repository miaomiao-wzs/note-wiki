# 销售退货物料移动类型配置

## 来源信息
- 来源类型：markdown
- 原始位置：`raw/sd/article/202607/销售退货物料移动类型配置.md`
- 适用模块：SD / MM / QM

## 一句话结论
销售退货移动类型不是只由订单类型决定，而是由销售订单类型、物料项目类别组、行项目类别、MRP 类型和计划行类别层层确定，最终在 `VOV6` 的计划行类别中控制移动类型。

## 示例差异
| 场景 | 恒生（3210） | 浙江康莱特（3130） |
| --- | --- | --- |
| 销售订单类型 | `ZRE1` | `ZRE1` |
| 普通项目类别组 | `NORM` | `NORM` |
| 行项目类别 | `ZRE3` | `ZRE1` |
| 物料 MRP 类型 | `PD` | `M0` |
| 计划行类别 | `ZD` | `Z3` |
| 物料移动类型 | `657` 退货冻结 | `655` 退货质检 |

## 配置链路
1. 用 `MM03` 查看物料“销售：销售组织数据 2”视图中的普通项目类别组。
2. 在后台配置中根据销售订单类型 + 普通项目类别组确定行项目类别。
3. 根据行项目类别 + MRP 类型确定计划行类别。
4. 用 `VOV6` 查看计划行类别中的交货冻结、移动类型等业务数据。
