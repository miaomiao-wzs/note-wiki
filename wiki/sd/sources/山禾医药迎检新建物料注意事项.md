# 山禾医药迎检新建物料注意事项

## 来源信息
- 来源类型：markdown
- 原始位置：`raw/sd/article/202607/山禾医药迎检新建物料注意事项.md`
- 适用模块：SD / MM / QM

## 一句话结论
迎检场景下如果需要新建物料并用于销售业务，必须补齐物料销售视图、交货工厂、物料科目分配组、运输 / 装载 / 可用性检查字段，并维护批次拆分策略。

## 操作要点
1. 用 `MM02` 维护销售组织 1 视图中的“交货工厂”，例如江西山禾对应工厂 `3500`。
2. 用 `MM02` 维护销售组织数据 2 视图中的“物料科目分配组”，示例值为 `10`。
3. 在销售：一般/工厂视图维护运输组、装载组、可用性检查等字段。
4. 用 `VCH2` 维护物料批次拆分策略，使交货单拣配时能自动批次拆分。
