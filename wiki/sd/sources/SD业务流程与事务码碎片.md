# SD业务流程与事务码碎片

## 来源信息
- 来源类型：markdown
- 原始位置：`raw/sd/article/202607/业务流程.md`
- 适用模块：SD / MM / FICO / QM

## 一句话结论
该资料是 SD 相关事务码和业务碎片清单，适合作为查询入口；后续应按主题拆分到销售订单、交货、开票、客户资质、物料扩充、库存查询等实体页和概念页。

## 重点事务码分类
### 销售订单与合同
- `VA01`：创建销售订单。
- `VA02`：修改销售订单。
- `VA03`：显示销售订单。
- `VA05`：销售订单清单。
- `VA11` / `VA21`：询价 / 报价。
- `VA31` / `VA41`：计划协议 / 合同。

### 交货与发货
- `VL10A`、`VL10C`：批量创建交货单。
- `VL06O`、`VL06F`、`VL06G`：交货清单和监控。
- `VL02N`：修改交货单 / 过账发货或退货收货。
- `ZSDR003`：销售发货明细表。

### 主数据与资质
- `BP`：查看或维护业务伙伴。
- `ZMMU010`、`ZSDU005`：扩充客户公司代码或销售视图。
- `ZMMU019`：扩充物料视图。
- `ZQMU002`、`ZQMU009`、`ZQMU011`：客户资质相关维护。

### 库存与批次
- `MMBE`、`MB52`、`MB5B`：库存查询。
- `MSC3N`：物料批次信息。
- `MD04`、`CO09`：物料需求和可用量检查。
