# 金税发票冲销作废

## 来源信息
- 来源类型：article
- 原始位置：`raw/fico/article/202607/金税发票冲销作废操作手册.docx`
- 语言：中文
- 适用模块：FICO / SD

## 一句话结论
金税发票冲销作废手册说明了线下作废或红冲金税发票后，如何通过 `ZFIR043`、`VF11`、`VF02`、`VF01` 修正形式发票和会计凭证。

## 关键内容
- 金税发票作废时，先线下作废金税发票，获取新的金税发票号和发票代码。
- 使用 `ZFIR043` 将新的金税发票信息更新到对应形式发票关系中。
- 金税发票红冲时，需要在形式发票上体现“一正一负一正”。
- 先线下红冲金税发票并取得新的金税发票号。
- 使用 `VF11` 冲销错误形式发票。
- 使用 `VF02` 对冲销形式发票发布到会计核算，生成冲销会计凭证。
- 使用 `VF01` 重新开票，再用 `VF02` 发布到会计核算生成正确正向凭证。
- 最后通过 `ZFIR043` 维护新开形式发票与新金税发票对应关系。
- 跨月问题发票通常将形式发票冲销到上月，正确发票做到当月；跨年问题需联系财务修改账期后处理。
