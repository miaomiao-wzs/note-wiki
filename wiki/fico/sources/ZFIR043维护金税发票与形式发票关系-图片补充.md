# ZFIR043维护金税发票与形式发票关系-图片补充

## 来源信息
- 来源类型：markdown
- 原始位置：`raw/fico/article/202607/ZFIR043维护金税发票与形式发票关系.md`
- 适用模块：FICO / SD

## 一句话结论
`ZFIR043` 用于维护金税发票与系统形式发票之间的对应关系，操作时先输入公司代码，再进入维护金税发票界面，根据是否已通过 `ZFIU026` 开通金税发票选择新增或修改，最后填写并保存对应关系。

## 操作步骤
1. 通过事务码 `ZFIR043` 进入初始界面，填写公司代码。
2. 点击“维护金税发票（系统发票）”按钮。
3. 填写公司代码和系统形式发票号。
4. 如果未使用 `ZFIU026` 开通金税发票，操作模式选择新增；否则选择修改。
5. 执行后填写对应信息，点击旁边灰色单元格使数据生效，再保存。

## 注意点
- 该 Markdown 主要补充截图附件；正文与既有操作手册存在重合。
- 浙江康莱特场景下，若维护金税票与形式票对应关系时出现税额与不含税金额差异，可能采用调整凭证统一处理，但后续对账时定位到单笔发货收入会更困难。
