# 金税发票红冲

## 概念定义
金税发票红冲是对已开金税发票进行红字冲销，并重新开具正确发票的处理。

## 为什么重要
- 红冲场景需要在形式发票上体现一正一负一正，并确保会计凭证同步。

## 工作机制 / 处理流程
1. 线下红冲金税发票。
2. 使用 `VF11` 冲销错误形式发票。
3. 使用 `VF02` 发布冲销会计凭证。
4. 使用 `VF01` 重新开票并发布会计。
5. 使用 `ZFIR043` 维护新关系。
