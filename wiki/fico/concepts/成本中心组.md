# 成本中心组

## 概念定义
成本中心组是把多个成本中心组织成层次结构的集合。

## 为什么重要
- 成本中心组便于批量查询、分配分摊、报表筛选和管理口径维护。

## 工作机制 / 处理流程
1. 使用 `KSH1` 创建成本中心组。
2. 按区间或单个成本中心加入组。
3. 使用 `KSH2` 维护结构。
4. 使用 `KSH3` 查询结构。
