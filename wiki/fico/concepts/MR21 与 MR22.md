# MR21 与 MR22

## 概念定义
MR21 与 MR22 是两个物料价格/价值调整工具，前者偏价格调整，后者偏库存总价值调整。

## 为什么重要
- 混淆两者会导致库存价值、物料分类账差异和实际成本分析出现偏差。

## 工作机制 / 处理流程
1. 使用 `MR21` 调整物料价格。
2. 使用 `MR22` 调整库存总价值。
3. 根据 S 价、V 价和物料分类账状态判断影响。
4. 用 CKM3/CKMLCP 分析后续差异。
