# CO 分摊分配配置说明

## 一、分摊概述

### 1.1 分摊 vs 分配的区别

| 特性     | 分摊（Assessment）   | 分配（Distribution） |
| -------- | -------------------- | -------------------- |
| 成本要素 | 使用**次级成本要素** | 保留**初级成本要素** |
| 明细程度 | 合并后分摊           | 按原始科目分摊       |
| 适用场景 | 不需要追溯原始科目   | 需要保留原始费用明细 |

### 1.2 相关事务码

| 事务码 | 功能         |
| ------ | ------------ |
| KSU1   | 创建分摊循环 |
| KSU2   | 修改分摊循环 |
| KSU3   | 显示分摊循环 |
| KSU4   | 删除分摊循环 |
| KSU5   | 执行分摊循环 |

---

## 二、分摊循环结构

### 2.1 循环层级

```text
分摊循环（Cycle）
├── 段1（Segment）
│   ├── 发送方（Sender）
│   └── 接收方（Receiver）
├── 段2（Segment）
│   ├── 发送方（Sender）
│   └── 接收方（Receiver）
└── ...
```

### 2.2 循环字段说明

| 字段         | 含义         | 填写说明            |
| ------------ | ------------ | ------------------- |
| **循环**     | 循环编号     | 自定义编号，如 Z001 |
| **起始日期** | 循环生效日期 | 一般填当月1日       |
| **文本**     | 循环描述     | 简要说明分摊用途    |

---

## 三、段（Segment）配置

### 3.1 段基本字段

| 字段               | 含义         | 可选值                         |
| ------------------ | ------------ | ------------------------------ |
| **段名**           | 段编号       | 如 S1、S2                      |
| **文本**           | 段描述       | 说明本段分摊的内容             |
| **发送者规则**     | 费用来源规则 | 可调整金额/百分比/固定金额     |
| **发送者追踪因子** | 分摊依据     | 统计指标/成本要素/实际作业     |
| **接收者规则**     | 接收方规则   | 可调整金额/固定金额/固定百分比 |

### 3.2 发送方规则

| 规则           | 说明                 |
| -------------- | -------------------- |
| **可调整金额** | 按实际发生的金额分摊 |
| **百分比**     | 按指定百分比分摊     |
| **固定金额**   | 按固定金额分摊       |

---

## 四、接收方追踪因素配置

### 4.1 追踪因子类型

| 类型           | 说明                       | 适用场景                     |
| -------------- | -------------------------- | ---------------------------- |
| **统计指标**   | 按统计指标数量比例分摊     | 按人数、面积、电脑台数等分摊 |
| **成本要素**   | 按特定成本要素金额比例分摊 | 按某类费用金额比例分摊       |
| **实际作业**   | 按实际作业消耗量比例分摊   | 按机器工时、人工工时等分摊   |
| **固定百分比** | 手动指定分摊比例           | 固定比例分摊                 |

### 4.2 统计指标配置

#### 创建统计指标

**事务码：KK01**

| 字段     | 含义      | 示例                                    |
| -------- | --------- | --------------------------------------- |
| 统计指标 | 编号      | HEAD                                    |
| 名称     | 描述      | 员工人数                                |
| 单位     | 计量单位  | 人、平方米、小时                        |
| 类别     | 固定/可变 | 固定（不随期间变化）/可变（期间会变化） |

#### 常用统计指标

| 指标               | 用途           |
| ------------------ | -------------- |
| 人员数（HEAD）     | 按人数分摊     |
| 面积（AREA）       | 按面积分摊     |
| 机器小时（MACHHR） | 按机器工时分摊 |
| 电脑台数（PC）     | 按电脑数量分摊 |

#### 录入统计数据

**事务码：KB31N**

| 字段     | 含义                 |
| -------- | -------------------- |
| 成本中心 | 统计数据所属成本中心 |
| 统计指标 | 指标编号             |
| 期间     | 数据期间             |
| 数量     | 指标数值             |

### 4.3 实际作业配置

#### 配置说明

当接收方追踪因素类型选择“实际作业”时：

- 系统会根据接收方成本中心实际消耗的作业类型数量来分摊费用
- 实际作业数量来自于成本中心实际接收到的作业类型消耗

#### 分摊计算公式

```text
接收方分摊金额 = 发送方费用 × (该接收方实际作业量 ÷ 所有接收方实际作业量总和)
```

#### 示例

| 接收方成本中心 | M10实际作业量 | 分摊比例 | 分摊金额 |
| -------------- | ------------- | -------- | -------- |
| 生产车间A      | 100小时       | 33.3%    | 3,333元  |
| 生产车间B      | 200小时       | 66.7%    | 6,667元  |
| **合计**       | **300小时**   | 100%     | 10,000元 |

#### 前提条件

1. **接收方成本中心必须有作业类型的消耗记录**
   - 来自生产订单确认（CO11N）
   - 来自作业分配（KB21N）

2. **作业类型必须已维护价格**
   - 事务码：KP26 - 维护作业类型计划价格

#### 适用场景

- 按机器工时分摊制造费用
- 按实际产出分摊辅助生产费用
- 需要反映实际生产负荷的分摊

#### 注意事项

| 情况                   | 影响                     |
| ---------------------- | ------------------------ |
| 接收方无作业消耗       | 该接收方分摊金额为0      |
| 所有接收方都无作业消耗 | 分摊失败，无金额被分摊   |
| 作业消耗只在部分期间有 | 只有有消耗的期间才能分摊 |

## 五、执行分摊

### 5.1 段锁定功能

若分摊循环中某个段当月无需分摊，可以**锁定该段**，正常执行其他段的分摊循环。

#### 操作方法

**事务码：KSU2** - 修改分摊循环

1. 进入 KSU2，输入循环编号
2. 选择需要锁定的段
3. 在段的**基本数据**页签，勾选**锁定**标识
4. 保存

#### 锁定效果

| 状态       | 执行效果             |
| ---------- | -------------------- |
| **未锁定** | 正常执行分摊         |
| **已锁定** | 跳过该段，不执行分摊 |

#### 使用场景

- 某些费用当月无需分摊（如季节性费用）
- 调试时临时跳过某个段
- 某些接收方当月无业务发生

### 5.2 执行事务码：KSU5

| 字段     | 含义                                   |
| -------- | -------------------------------------- |
| 循环     | 输入创建的循环编号                     |
| 起始日期 | 循环生效日期                           |
| 测试运行 | 勾选先测试，确认无误后取消勾选正式执行 |
| 明细清单 | 勾选查看分摊明细                       |

### 5.3 查看分摊结果

| 事务码         | 说明                      |
| -------------- | ------------------------- |
| S_ALR_87013611 | 成本中心实际/计划对比报表 |
| KSB1           | 成本中心行项目报表        |
| KP27           | 显示作业类型价格          |

## 七、常见问题

### Q1：分摊执行后没有金额？

- 检查发送方成本中心是否有费用
- 检查接收方是否有追踪因子数据（统计指标/作业消耗）
- 检查循环的起始日期是否正确

### Q2：分摊比例不对？

- 检查统计指标数据是否录入正确
- 检查作业类型消耗是否准确
- 检查接收方追踪因素配置是否正确

### Q3：如何撤销分摊？

- 事务码：KSU5 执行时勾选“冲销”选项
- 或使用 FB08 冲销产生的会计凭证
