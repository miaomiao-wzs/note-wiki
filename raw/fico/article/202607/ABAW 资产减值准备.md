# SAP 固定资产减值准备配置与实操指南

> **适用场景**：固定资产账面价值高于可收回金额时，需计提减值准备。
> **SAP版本**：S/4HANA（ECC 6.0 基本通用）
> **关键事务码**：ABAW, AB08, AO84, AO90, OABW, OAYR, AFAB, AW01N


## 一、减值处理方式对比

| 处理方式 | 事务码 | 核心逻辑 | 优缺点 |
| :--- | :--- | :--- | :--- |
| **资产重估减值（推荐）** | `ABAW` | 直接调整资产账面净值，生成减值准备 | ✅ 标准方法，更新资产卡片<br>✅ 子账与总账一致<br>❌ 需后台配置 |
| **计划外折旧** | `ABAA` | 作为特殊折旧处理，冲减净值 | ✅ 操作灵活<br>❌ 本质仍为折旧，科目调整受限 |
| **手工总账凭证（不推荐）** | `F-02` / `FB50` | 直接录入会计凭证 | ✅ 简单直接<br>❌ 不更新资产子账，易导致账账不平 |

### 2.4 分配总账科目（AO90）

| 步骤 | 事务码 | 操作说明 |
| :--- | :--- | :--- |
| **进入科目确定** | `AO90` | 路径：`IMG → 财务会计 → 资产会计核算 → 与总账会计集成 → 分配总账科目` |
| **选择资产分类** | `AO90` | 找到对应资产分类的“科目确定”条目（如 `0001`）并双击进入。 |
| **配置重估科目** | `AO90` | 在 **“重估价 (Revaluation on APC)”** 栏目中，为新事务类型（如 `Z92`）配置：<br>• **借方 (Dr.)**：`固定资产减值损失`（损益科目）<br>• **贷方 (Cr.)**：`固定资产减值准备`（资产负债科目） |

> **配置验证**：若 `ABAW` 报错 `AA629`（无适合资产价值的交易类型）或 `AA816`，通常是因为 `AO90` 未分配科目，请返回检查。

## 三、前台操作流程

### 3.1 标准减值操作（ABAW）

| 步骤 | 事务码 | 操作说明 |
| :--- | :--- | :--- |
| **进入事务** | `ABAW` | 输入公司代码。 |
| **选择事务类型** | `ABAW` | **关键**：当年购置的资产选 `Z92`（或系统标准），以前年度的选 `Z93`。 |
| **填写资产信息** | `ABAW` | 输入资产编号（如 `001050015333`）、过账日期、资产价值日（须大于资本化日期）。 |
| **输入减值金额** | `ABAW` | 在“记账金额”字段输入**减值金额**（如账面100万，可收回80万，则输入20万）。 |
| **保存** | `ABAW` | 点击保存，生成资产重估凭证。 |

> **注意**：`ABAW` 仅更新资产子模块，**总账凭证在月底运行 `AFAB`（折旧）时才会生成**，分录为：
> - 借：固定资产减值损失
> - 贷：固定资产减值准备

### 3.2 冲销错误凭证（AB08）

当误操作导致价值增加或方向错误时：

| 步骤 | 事务码 | 操作说明 |
| :--- | :--- | :--- |
| **进入事务** | `AB08` | 输入公司代码。 |
| **定位凭证** | `AB08` | 点击“其他凭证”，输入错误凭证号，系统自动识别。 |
| **确认冲销** | `AB08` | 选择冲销原因（如 `01` 红冲），检查后保存。 |


## 四、常见报错与处理

| 报错消息号               | 报错文本                   | 原因分析                         | 解决方案                                           |
| :------------------ | :--------------------- | :--------------------------- | :--------------------------------------------- |
| **AA629**           | 输入适合资产价值的交易类型          | 所选事务类型与资产资本化年份不匹配，或折旧范围禁止负值。 | 检查使用 `Z92`/`Z93` 是否正确；在资产卡片折旧范围中勾选“负值允许”。      |
| **AA816**           | 事务类型 XXXX 未对重估定义       | `AO90` 中未为新事务类型分配总账科目。       | 返回 `AO90`，配置该事务类型的借方/贷方科目。                     |


## 五、验证方法

| 验证点 | 事务码 | 检查内容 |
| :--- | :--- | :--- |
| **资产净值** | `AW01N` | 账面净值是否**减少**了减值金额。 |
| **价值明细** | `AW01N` | 在“过账值”页签 → “重估”栏目，查看**负数金额**记录。 |
| **变更历史** | `AS03` | 菜单 `环境 → 更改`，查看完整的资产价值调整轨迹。 |
| **折旧计划** | `AW01N` | 检查次月起的计划折旧额是否已按减值后净值重新计算。 |
| **总账凭证** | `FB03` | 运行 `AFAB` 后，检查生成的会计凭证：借：资产减值损失；贷：减值准备。 |


## 六、操作口诀

> **减值不离 ABAW，当年 Z92 往年 Z93；**
> **AO90 配好科目，AFAB 跑完凭证出；**
> **若错用 AB08 冲，方向对了再重复。**


## 七、实施检查清单

在正式环境操作前，请确认以下事项：

- [ ] `OABW` 中折旧范围已启用“重估”
- [ ] `OAYR` 中公司代码已开启重估过账
- [ ] `AO84` 中已存在 `Z92` / `Z93` 事务类型
- [ ] `AO90` 中已为上述类型分配了正确的总账科目
- [ ] 已在**测试系统**完整模拟过 `ABAW` → `AB08` → `ABAW` 流程
- [ ] 已确认减值金额不会导致资产净值变为负数
- [ ] 已与财务团队确认减值使用的损益科目和备抵科目

> **最后提醒**：减值准备一旦计提，**通常不可转回**（按中国会计准则）。操作前务必与业务及财务负责人确认金额与合理性。生产环境操作需走变更审批流程。
