# SAP 生产工单收货数量与质检取样成本

**日期**：2026-06-11  
**分类**：工作硬实力  
**标签**：#SAP #PP #CO #QM #生产工单 #工单收货 #质检取样 #成本核算 #医药行业

---

## 原始问题

用户询问：

> 生产工单计算得率时，是按照质检放行后的数量来计算的。此时工单收货数量是不是应该扣减生产取样后的数量？收货库存总金额是工单成本吗？还是会把生产质检取样后的检验检测费也包含进去？

这个问题表面上是在问“得率”，实际包含三个口径问题：

1. **工单收货数量口径**：生产订单实际应该收货多少？
2. **质检放行数量口径**：最终通过质量检验、可以销售或使用的数量是多少？
3. **库存金额和成本口径**：工单收货库存金额是否等于工单实际成本？质检取样成本应该计入库存成本还是检验检测费？

---

## 核心结论

**生产订单收货数量、质检放行数量、得率计算数量是三个不同口径，不能混在一起。**

一句话理解：

```text
工单收货数量 = SAP 实际 101 收货过账数量
质检放行数量 = 检验合格后可使用 / 可销售的数量
得率计算数量 = 企业管理制度定义的计算口径
```

再补充成本判断：

```text
库存金额不一定等于工单实际成本。
标准价物料通常按标准成本入库，工单实际成本与入库金额之间的差异通过订单结算处理。
```

质检取样成本是否进入库存成本，要看取样发生时点和企业会计政策：

```text
完工前取样：更容易表现为生产过程损耗或生产成本
完工后取样：更容易表现为质量检验消耗或检验检测费
```

---

## 当前流程：生产过程中留样，扣减后收货

### 业务流程

当前用户描述的流程是：

```text
生产工单下单数量：1000 个
生产过程中留样：50 个
工单实际收货：950 个
工单成本：1000 元
```

即：生产过程中把待包装的半成品留出一部分给质量检验，工单收货时扣除留样数量，只对剩余数量收货。

### 系统结果

如果 SAP 中只对 950 个做生产订单收货，那么系统里的库存数量就是：

```text
工单收货数量 = 950 个
库存增加数量 = 950 个
```

### 成本含义

如果整个工单实际投入成本是 1000 元，但最终只收货 950 个，那么这 50 个取样数量的成本没有单独作为质量检验费用体现，而是被剩余 950 个入库产品承担，或者体现在生产订单差异中。

从管理含义上看：

```text
生产实际投入服务于 1000 个
系统最终只入库 950 个
50 个取样被隐藏为生产过程损耗 / 生产成本 / 工单差异
```

如果简单按实际成本理解，单位成本会被抬高：

```text
1000 元 / 950 个 = 1.0526 元/个
```

但在 SAP 标准价场景下，不能直接这样认定库存金额，仍然要看物料价格控制和订单结算逻辑。

---

## 当前流程的问题

当前流程的优点是操作简单：生产过程中直接留样，最终少收货，省去了后续从仓库取样、拆箱、领用、费用过账等操作。

但它存在几个问题。

### 1. 生产产出数量被低估

实际生产可能完成了 1000 个，但系统只体现 950 个收货。

这样会导致系统中看不清：

```text
生产完成数量是多少？
质检取样消耗是多少？
真正生产损耗是多少？
最终可销售数量是多少？
```

生产损耗和质检取样被混在一起。

### 2. 质检取样成本不透明

50 个样品的成本没有单独进入质量成本或检验检测费，而是隐藏在生产成本或生产订单差异中。

这会影响责任归属：

```text
这是生产损耗？
还是质量管理部的检验消耗？
应该归生产部门？
还是归质量管理部门？
```

### 3. 质检对象可能不是最终成品

当前流程是在生产过程中对待包装半成品留样。

对于医药行业，最终产品质量不仅包括内容物，还包括：

- 外包装是否完整；
- 标签是否正确；
- 批号是否准确；
- 效期是否正确；
- 说明书是否匹配；
- 封口、装盒、装箱是否符合要求；
- 追溯信息是否完整。

如果在包装完成前取样，质检对象就不是最终销售状态产品。

这会导致：

```text
检验对象 ≠ 最终销售产品
```

从质量合规角度看，这个流程不够严谨。

## 更合理流程：先完整收货，再由质量部门取样

### 建议流程

更清晰、更合规的流程是：

```text
生产工单下单数量：1000 个
生产实际完成数量：1000 个
工单收货入库数量：1000 个
质量部门从库存取样：50 个
最终可销售 / 可放行数量：950 个
```

即：

1. 生产部门先按实际生产完成数量做工单收货；
2. 产品进入质检状态或库存；
3. 质量管理部门从库存中取样 50 个进行检验；
4. 取样消耗通过库存发货、样品消耗、报废或质量费用过账单独体现；
5. 检验合格后，剩余数量转为可销售或可用库存。

### 管理效果

这样可以清楚区分：

```text
生产产出：1000 个
质检取样：50 个
质量放行 / 可销售数量：950 个
```

不会把质检取样误认为生产没有完成，也不会把质量检验消耗隐藏在生产损耗中。

## 工单收货数量、质检放行数量、得率计算数量的区别

这是本问题最重要的口径区分。

### 工单收货数量

```text
工单收货数量 = 生产完成并在 SAP 中执行 101 收货过账的数量
```

如果收货 950，系统库存增加 950。  
如果收货 1000，系统库存增加 1000。

### 质检放行数量

```text
质检放行数量 = 质量检验合格后允许使用或销售的数量
```

如果收货 1000，取样消耗 50，剩余 950 合格放行，则：

```text
质检放行数量 = 950
```

### 得率计算数量

```text
得率计算数量 = 企业管理制度定义的口径
```

不同企业可能有不同定义。

如果得率定义为生产产出得率：

```text
生产产出得率 = 工单收货数量 / 理论产出数量
             = 1000 / 1000
             = 100%
```

如果得率定义为质量放行得率：

```text
质量放行得率 = 质检放行数量 / 工单下单数量
             = 950 / 1000
             = 95%
```

因此不能因为“得率按质检放行数量算”，就反推“工单收货数量必须扣减取样数量”。

更准确的表达是：

> 工单收货数量反映生产完成数量；质检放行数量反映质量合格数量；得率按哪个数量算，要看企业管理制度定义。

## 收货库存金额是否等于工单成本

不能直接回答“等于”。

SAP 中生产订单收货金额取决于以下因素：

```text
物料价格控制：S 标准价 / V 移动平均价
生产订单实际成本
标准成本估算
订单结算规则
是否启用物料分类账 / 实际成本核算
```

### 标准价 S 场景

很多成品、半成品使用标准价。

生产订单收货时通常按标准成本入库：

```text
借：库存 = 收货数量 × 标准价
贷：生产订单
```

例如：

```text
标准价：1 元/个
收货数量：950 个
入库金额：950 元
```

如果工单实际成本是 1000 元，那么差额：

```text
1000 - 950 = 50 元
```

通常不会直接进入库存金额，而是作为生产订单差异，在订单结算时进入差异科目，或在启用物料分类账 / 实际成本核算时再进行分摊。

所以标准价场景下，不能简单说：

```text
收货库存金额 = 工单实际成本
```

更准确的说法是：

> 生产订单收货时库存通常按标准成本入库，工单实际成本与入库金额之间的差异通过订单结算和差异处理体现。

### 移动平均价 V 场景

如果物料使用移动平均价，生产收货金额可能更接近实际成本逻辑，但仍要看具体配置、订单结算和企业核算方式。

### 启用物料分类账 / 实际成本核算

如果企业启用 Material Ledger / Actual Costing，期末可能会把实际成本差异重新分摊到库存和消耗中。

因此最终库存金额可能在实际成本结算后发生调整。

---

## 质检取样成本是否进入库存成本

判断原则是：看取样发生在生产完工前还是完工入库后，以及企业会计政策如何定义。

### 场景一：生产过程中的必要检验

如果检验发生在生产过程中，是把产品生产到当前状态所必需的一部分，例如：

- 生产线过程检验；
- 半成品检验；
- 工艺路线中的检验工序；
- 检验工时或检测资源作为作业类型计入工单；
- 正常生产过程中的质量控制。

那么相关成本更可能作为生产成本的一部分计入工单成本。

逻辑是：

```text
检验活动是生产过程的一部分
因此检验成本可以计入产品成本
```

### 场景二：完工入库后的质量抽检

如果产品已经完工入库，质量部门从库存中取样检验，且样品被破坏、消耗、不能销售，那么这更像质量检验费用或质量成本。

可能的会计处理逻辑是：

```text
借：检验检测费 / 质量成本中心 / 质量费用科目
贷：库存
```

此时 50 个样品的成本转入质量费用，不再留在库存中。

### 场景三：检验后仍可销售

如果取样后样品没有被破坏，仍然可以重新入库销售，则不应直接进费用。

应考虑：

- 质检库存；
- 冻结库存；
- 样品返库；
- 检验合格后转非限制使用库存；
- QM 状态管理。

## 当前流程与建议流程对比

| 维度 | 当前流程：生产中留样后只收 950 | 建议流程：先收 1000，再质检取样 50 |
|---|---|---|
| 工单收货数量 | 950 | 1000 |
| 生产产出反映 | 容易低估生产完成数量 | 更准确反映生产完成数量 |
| 取样数量体现 | 被隐藏为生产损耗或工单差异 | 单独体现为质量取样 |
| 成本归属 | 50 个成本由剩余产品承担或进入生产差异 | 50 个成本可转入检验检测费 / 质量成本 |
| 质量检验对象 | 可能是待包装半成品 | 可以是最终包装完成的成品 |
| 外包装检验 | 可能无法覆盖 | 可以覆盖 |
| 管理清晰度 | 生产、质检、库存责任混在一起 | 生产、质检、库存责任更清晰 |
| 合规性 | 省事但质量控制偏弱 | 更符合最终产品放行逻辑 |

## 医药行业的特殊关注点

医药行业中，生产完成、检验完成、质量放行、可销售不是同一个概念。

可能存在以下状态链路：

```text
生产完工
工单收货
进入质量检验库存
检验中
检验合格
质量放行
转非限制使用库存
可销售
```

因此更专业的 SAP/QM 处理思路是：

> 工单可以先按实际生产完成数量收货进入质检库存，待质量检验和质量放行后，再将合格数量转为非限制使用库存；取样消耗则通过样品消耗、库存发货、报废或质量费用领用单独体现。

对于医药行业，外包装、标签、批号、效期、追溯信息也属于质量控制的重要对象。

如果在包装前进行留样检验，可能无法覆盖最终销售状态产品的质量要求。

## 当前暴露的知识盲区

### 盲区一：容易混淆三种数量口径

之前容易把以下三个数量混成一个问题：

```text
工单收货数量
质检放行数量
得率计算数量
```

后续需要先问清楚用户到底在问哪一个口径。

### 盲区二：对生产订单收货金额理解不够精确

不能直接说“收货库存金额就是工单成本”。

应先确认：

```text
物料价格控制是 S 还是 V？
工单是否按标准成本收货？
订单差异如何结算？
是否启用物料分类账 / 实际成本核算？
```

### 盲区三：取样成本的系统处理路径还需要补查

“50 个进检验检测费”方向可以，但系统上需要明确动作。

可能方式包括：

```text
从非限制库存发货到质量成本中心
从质检库存转报废
通过移动类型做样品消耗
通过 QM 检验批样品管理记录消耗
通过成本中心领用进入检验检测费
```

后续应结合企业实际配置确认移动类型、成本中心和会计科目。

### 盲区四：质量检验费用和产品成本的边界需要明确

判断原则：

```text
为了把产品生产到可销售状态所必需的检验成本
更可能进入产品成本

产品完工入库后，为验证、抽检、监督发生的成本
更可能作为质量费用或检验检测费
```

但最终仍要服从企业会计政策和审计口径。

### 盲区五：医药行业需要结合 QM 状态和 GMP 要求

医药行业不是简单地“生产完成就可销售”。

需要考虑：

- 检验批；
- 质检库存；
- 质量放行；
- 批次状态；
- 包装标签检查；
- 可销售状态转换；
- GMP 合规要求。

## 给业务用户的标准解释

可以这样回答用户：

> 收货数量要看企业定义的业务流程。如果生产过程中已经把取样数量扣掉，只对剩余数量做工单收货，那么 SAP 中的工单收货数量就是扣减取样后的数量。例如生产 1000 个、留样 50 个，最终只收货 950 个。这样 50 个样品的成本会被吸收到剩余入库产品成本或生产订单差异中，管理上不容易单独看出质检取样消耗。  
>   
> 更清晰、更合理的做法是，生产工单按实际完成数量 1000 个收货，进入质检状态或库存，然后由质量部门从库存中取样 50 个做检验。这样系统可以清楚地区分生产产出 1000、质检取样 50、最终放行或可销售数量 950。取样的 50 个如果被消耗或破坏，可以通过库存发货、样品消耗或报废等方式转入检验检测费或质量成本。  
>   
> 至于库存金额是否等于工单成本，要看物料价格控制和成本核算方式。标准价物料通常按标准成本入库，工单实际成本和入库金额之间的差异在订单结算时处理；如果启用实际成本核算，期末可能再分摊差异。  
>   
> 从医药行业质量管理角度看，如果只在包装前留样，可能检验不到最终外包装、标签、批号等是否合规。更严谨的方式是对最终包装完成后的产品进行取样检验，再根据检验结果放行库存。

## 后续需要补查的问题

1. 当前成品 / 半成品物料的价格控制是 S 还是 V；
2. 生产订单收货是否按标准成本入库；
3. 工单实际成本与收货金额差异如何结算；
4. 是否启用物料分类账 / 实际成本核算；
5. 质检取样在 SAP 中目前用什么移动类型处理；
6. 取样消耗的成本中心和费用科目是什么；
7. 样品检验后是否全部破坏，还是存在返库情况；
8. QM 中是否启用检验批、质检库存、使用决策、批次状态管理；
9. 医药行业质量文件中对取样时点是否有明确规定；
10. 得率指标的企业定义到底是生产产出得率、质量放行得率，还是综合有效产出率。

---

## 一句话沉淀

> 生产订单收货数量反映生产完成，质检放行数量反映质量合格，得率计算数量取决于管理口径；质检取样如果发生在完工入库后，应单独体现为质量检验消耗，而不是隐藏在生产收货扣减中。
